glow-card

Cursor-tracking glow border cards. Zero dependencies. <2KB gzipped.

Individual Cards

Border Glow

The classic cursor-tracking glow border. Uses variant="border" (default).

Background Glow

Subtle background illumination that follows your cursor. Uses variant="background".

Spotlight

Focused spotlight beam effect. Uses variant="spotlight".

Rainbow Border

Rotating rainbow gradient border. Uses variant="rainbow".

Glow Line

Rotating glow line effect. Uses variant="glow-line".

Pulse Glow

Pulsating glow with custom color. Uses variant="pulse".

Custom Colors

Fully customizable via CSS variables. Red glow, wider spread, thicker border.

Card Group (Proximity Glow)

Card A

Part of a group. Glow follows cursor across all cards simultaneously.

Card B

Adjacent cards glow with proximity-based intensity.

Card C

The "Stripe dashboard" effect with zero effort.
